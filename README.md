# cryptographicProtectionTools

Набор простых примеров на Java, демонстрирующих классические методы симметричного шифрования и элементарного криптоанализа.

## Содержимое

- **CaesarCipher** — шифр Цезаря с поддержкой латиницы и кириллицы.
- **VigenereCipher** — реализация шифра Виженера.
- **SubstitutionCipher** — моноалфавитная подстановка для русского алфавита.
- **StreamCipher** — примитивное гаммирование на XOR с псевдослучайной последовательностью.
- **KasiskiTest** — тест Касиски для оценки длины ключа шифра Виженера.

Все классы содержат `main`-методы с демонстрацией работы и могут запускаться независимо.

## Сборка и запуск

Для компиляции требуется установленный JDK (версия 8 или новее):

```bash
javac -encoding UTF-8 -d bin src/*.java
```

Запуск примера (например, шифра Цезаря):

```bash
java -cp bin CaesarCipher
```

Аналогично можно запускать `VigenereCipher`, `SubstitutionCipher`, `StreamCipher` и `KasiskiTest`.

Проект предназначен для учебных целей.
